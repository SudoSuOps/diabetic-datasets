# DiabeticDatasets

*Open diabetic datasets, donated free for builders.* The data layer of the hive.

Over 1,000,000 honey datasets donated by **Swarm & Bee → OpenDiabetic**, offered free (no charge,
no strings) to developers and researchers. **Crystal clear:** every dataset shows its schema, real
sample rows, size, and SHA-256 hash before download — verify what's in the file yourself.

- Static site (no framework) for **diabeticdatasets.com** on Cloudflare Pages.
- `catalog.json` — the dataset catalog. `data/*.jsonl` — real downloadable samples.
- **No PHI** — open education + synthetic instruction sets only, never patient records. Not medical advice.

*OpenDiabetic is the hive · LocalDiabetic is the healers · DiabeticLedger is the proof · DiabeticDatasets is the cookbook.*
